## 1. 배경 / 갭

- 이사님 단기 목표(07-03): 신규 로직(비중 슬라이드) 후위 → **기존 웹을 라오어 이론 시각화·초등학생 레벨**.
- **갭:** 현재 `dashboard.py` = "전략 실험실"(전략·exit_ratio 강도 비교). 라오어 이론 **자체를 설명/체감시키진 않음**.
- → 기존 엔진(backtest/)은 재사용하되, 그 앞에 **라오어 이론을 아이 수준에서 시각화하는 "체험관"**을 추가.

## 2. 라오어 이론 핵심 (seminar-raoer 리뷰 정리)

아이에게 설명할 5문장:
1. 투자할 돈을 **작게 40조각**으로 나눈 뒤
2. **매일 조금씩 사**되,
3. **가격이 싼(하락한) 날일수록 더 많이 사**서
4. 내 **평균 매입가를 계속 낮추고**,
5. **안 팔고**(무한 매수) 버티면 **우상향장에서 복리**로 수확.
- 본질 엣지 = "무서울 때 더 사는 기계적 룰"(감정 이김). (세미나 §3 자동화 역설과 연결 — 교육 포인트)

## 3. 핵심 메타포 (제안)

> **"싸면 더 사자 — 평단가 미끄럼틀"**
- 일상 비유: 과자가 100원일 땐 10개, **50원으로 싸지면 20개** 사서 → 전체 과자의 **평균 가격을 낮춘다**.
- '평단가 미끄럼틀' = 매수할수록 평균 가격이 미끄러져 내려가는 이미지.
- ★ 이사님 안건: 과자값 말고 더 좋은 비유(용돈·게임 아이템·사탕 등) 선호 여부.

## 4. 화면 구성 (우선순위)

기존 `dashboard.py`에 **탭** 또는 스크롤 섹션으로 추가(실험실은 유지).

| # | 이름 | 내용 | 체감 포인트 |
|---|---|---|---|
| ① | 라오어가 뭔가요? | §2 5문장 + 과자 비유 일러스트 | 첫 진입 교육 |
| ② | 싸면 더 사자 | 가격 슬라이더 ↓ → 살 개수 ↑ 게이지/막대 (비중함수 체감) | "싸면 더 많이" 직관 |
| ③ | 평단가 미끄럼틀 | 매수 누적에 따라 평균가가 내려가는 계단/라인 애니메이션 | 핵심 엣지 시각화 |
| ④ | 진짜 시장에서 한 판 | 실제 QQQ 데이터 + 재생/슬라이더 → 시점별 매수·평단가·자산 실시간 | 이론↔실제 연결 |
| ⑤ | 왜 돈이 되나 | 복리 곡선 vs 사고보유(B&H) — 기존 실험실과 연결 | 수확 체감 |

**우선 구현 제안:** ①②③ 먼저(이론 핵심 체감), ④(실데이터)·⑤(복리)는 2차.

## 5. 기존 자산 재사용

- `backtest/engine.py` `run_backtest`·`metrics` — ④⑤용.
